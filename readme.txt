通过阅读并理解本书，那么在使用C#的时候会感觉到更加顺手，而且会更加自然地凭自己的直觉行事，而不会有太多的犹豫，你将拥有强大的自信吗，知道自己有哪些选择，并知道C#语言本身在鼓励你选择那条路走下去。

路线图

本书的结构很简单，共有5个部分和3个附录。
第一部分相当于简介，包括对C# 1重点主题的回顾，这些主题对于理解C#的后续版本非常重要，而且经常被人误解。
第二部分讲述了C# 2的新特性。
第三部分讲述了C# 3的新特性，等等


这样组织全书，意味着一个主题会被反复讲到------尤其是委托在C#2.0中有所增强，在C#3.0则进一步增强，但读者在不同的项目中使用不同版本的C#：
例如：在工作中使用C#4.0，在家使用C#5.0实验，
所以我们有必要明确那个版本有哪些内容，这样可以营造出一种现场感和进化感， 可以体会到随着时间的推移，语言是如何进化的。

第一章的作用
是搭建一个舞台，从一段简单的C#1.0代码开始，让它不断演变，观察更高的版本如何使代码变得越来越易读，越来越强大。我们
介绍C#成长的历史背景，以及它作为一个完整平台的一部分而工作的技术背景。
具体地说，C#作为一种语言，它的基础是各种各样的“框架库”（net framework中的各种库）以及一个强大的运行时（runtime）。借助它们，我们可以把抽象的东西转变成现实。
